# Matrix Multiplication C Baseline

This is a simple C11 baseline solution for the `matrix-multiplication` demo challenge.
It reads `input.bin` from `AGENTICS_INPUT_DIR`, writes `output.bin` to
`AGENTICS_OUTPUT_DIR`, and performs a straightforward row-major matrix
multiplication for each case in the batch.

The implementation is intentionally plain and portable. It is useful for smoke
testing the challenge protocol, not as a competitive optimized submission.
